# TRENZYCH PANEL

Modern VPN Management Panel for Xray, Outline, Shadowsocks and more.

## Features

- Modern Responsive UI
- Admin Dashboard
- Multi Server Support
- Live Server Status
- Real Ping Checker
- Copy Config
- Telegram Support Button
- Health Check
- Backup & Restore
- One Click Installer

## Requirements

- Ubuntu 22.04 / 24.04
- Apache2
- PHP 8+
- SQLite3

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/trenzych-v2.git
cd trenzych-v2
sudo ./scripts/install.sh
```
